# BaaS(Blockchain as a Service）

## BaaS 简介
BaaS 为 WaykiChain网络（mainnet/testnet）提供 HTTP API接口。
与以太坊平台上的 `Infura` 类似，BaaS 提供在运行在远程服务器的钱包节点，因此您无需自己部署一个节点接入WaykiChain工作。 以便在WaykiChain 上开发去中心化应用（DAPP）或者 钱包应用。
