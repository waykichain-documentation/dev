
## API error code  分析

* [-4],submittx Error:operate-minus-account-failed
  * 原因: WICC 余额不足
  * 解决方法: 请检查提交信息和余额


* [-4],submittx Error:not-sufficiect-funds
  * 原因: WICC 余额不足
  * 解决方法: 请检查提交信息和余额


* bad-read-accountdb
  * 原因: 输入的用户地址错误
  * 解决方法: 请检查源账户地址和目的账户地址


* submittx Error:bad-getaccount
  * 原因: 输入的regid错误
  * 解决方法: 请检查regid是否正确


* submittx Error:bad-signscript-check
  * 原因: 签名有问题
  * 解决方法: 请检查签名
